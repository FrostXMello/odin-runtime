# Cognitive Interface

Prompt 41 unified cognitive shell merges conversational state, workspace presence, and live UI adaptation.

## Modules

- `core/cognitive_shell/` — session orchestration, attention visualization, adaptive UI modes
- `core/conversation_runtime/` — streaming dialogue, memory threads, multi-mode conversation
- `core/presence/` — bounded emotional modeling with explicit simulation disclosure
- `core/cognitive_visualization/` — live reasoning graphs and thought streams
- `core/live_overlay/` — contextual workspace overlays (supervised modes)
- `core/self_development/` — supervised improvement proposals only
- `core/transparency/` — AI disclosure, autonomy boundaries, operator visibility

## Enable

```env
ODIN_COGNITIVE_SHELL_ENABLED=1
ODIN_CONVERSATION_RUNTIME_ENABLED=1
ODIN_PRESENCE_ENABLED=1
ODIN_COGNITIVE_VISUALIZATION_ENABLED=1
ODIN_LIVE_OVERLAY_ENABLED=1
ODIN_SELF_DEVELOPMENT_ENABLED=1
ODIN_TRANSPARENCY_ENABLED=1
ODIN_COGNITIVE_INTERFACE_MODE=balanced
```

Modes: `minimal`, `balanced`, `immersive`, `cinematic`.

## Safety

- No unrestricted autonomy
- Self-development proposes plans only — human approval required
- Emotional states are simulated and disclosed
- Local-first processing preserved
