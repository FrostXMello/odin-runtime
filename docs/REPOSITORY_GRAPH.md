# Repository Graph

Deep multi-file repository intelligence.

## Capabilities

- Repository topology mapping
- Semantic code map
- Dependency and service graphs
- Execution flow tracing
- API contract mapping
- Architecture drift detection

## API

- `POST /api/v1/runtime/repository-graph/analyze`

Operator page: `/runtime/repository-graph`

Trace: `architecture_drift_detected`, `repository_graph_updated`
