# Workstation Runtime

Real-time workstation awareness layer.

## Capabilities

- Live activity graph
- Semantic window analysis
- Application relationship mapping
- Desktop flow memory
- Privacy-preserving monitoring with exclusions

## API

- `GET /api/v1/runtime/workstation`
- `POST /api/v1/runtime/activity-graph/observe`

Trace: `workspace_attention_shifted` on `workstation:runtime`.
