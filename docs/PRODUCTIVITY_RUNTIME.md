# Productivity Runtime (v0.36)

Local-first task tracking, focus sessions, reminders, and workload analytics.

## Configuration

```env
ODIN_PRODUCTIVITY_ENABLED=true
ODIN_COMMUNICATIONS_ENABLED=true
```

## APIs

- `GET /api/v1/runtime/tasks`
- `POST /api/v1/runtime/tasks`
- `POST /api/v1/runtime/focus/start`
- `GET /api/v1/runtime/briefings`
